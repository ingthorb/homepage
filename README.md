# Home Page
Done with React.
Created to showcase portfolio and other information about my career.
Animation is connected to Weather API in Bern.

## Run Project
### 1. Clone the project

### 2. Run the project
```shell
npm i
npm start
```

### 3. Build
```shell
npm run build
```
